## a. 与 Git 同类型的工具有？

```
Git属于版本控制管理工具，同类型常用的有CVS, SVN 和 Mercurial
```

## b. Git 分了哪几个区？

```
Git可简单分为三个区:
1、工作区(working directory)
2、暂缓区(stage index) 
3、历史记录区(history)
```
